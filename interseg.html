INTERSEG: Interactive 3D Segmentation



Description

Fluorescence microscopy enables the visualization of cellular components through the use of fuorescent tags. Automated and reliable identification of single cells in stacks of microscopic images is necessary to quantify the expression levels of different transcription factors in tissues, whcih provides valuable information about the different progenitor cell subpopulations and their spacial relationships. To address this needs, we developed InterSeg and a user friendly interface.

InterSeg is a collection of java plugins in ImageJ for the semi-automatic 2D/3D segmentation of convex cells on tissue slices images, quantification of fluorescent staining and counting of cells in specific regions of interest (ROIs). The software can handle arbitrarily large images and efficiently detect the borders of single cells. Given the ability to integrate input in form of minimal user operations, we are able to segment cells even in difficult conditions such as poor image contrast, highly densely packed cells and different convex cell morphologies.

Our software has several advantages over previous implementations:

  • Use of a general segmentation algorithm, since no assumption on the cell shape, density or contrast is made.
  • Local algorithm, which can exploit multi-core machines to speed up image processing and scalable to any image size.
  • Possibility of user intervention during the identification of the cell locations.
  • User friendly interface by integration in ImageJ and full automatized process pipeline.
  • Open source code released under the GNU Public License.

Source code

The algorithm and results are described in Pfister et al, Robust 3D cell segmentation by local region growing in convex volumes, Biomedical Imaging (ISBI), 2013 IEEE.
